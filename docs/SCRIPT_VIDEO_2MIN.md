# 🎬 Script vidéo EGREENCITY'S — « Le projet en 2 minutes »

**Format** : motion design + plans réels Guyane
**Durée cible** : 2 min 00 s (± 5 s)
**Langue** : français (VO) — sous-titres français intégrés (malentendants + lecture muette sur mobile)
**Ratio** : 16:9 (YouTube + site web) + version 9:16 à rééditer pour Reels/TikTok
**Ton** : institutionnel chaleureux, rythmé, fierté locale
**Musique** : instrumentale mid-tempo, guitare acoustique + percussions légères (royalty-free : Epidemic Sound « Sunny Isles » ou équivalent)

## 🎤 Brief voice-over (à destination du comédien / IA)

- **Voix** : homme, 35-45 ans, accent neutre français métropolitain ou légèrement guyanais (préféré pour authenticité locale)
- **Débit** : 150-160 mots par minute (rythmé mais posé)
- **Intentions** :
  - Scènes 1-2 : grave, constat, engagement
  - Scènes 3-4 : technique, pédagogique, rassurant
  - Scènes 5-6 : visionnaire, fier
  - Scène 7 : chaleureux, invitation
- **Mots à appuyer** : *connecter*, *six communes*, *e-City*, *e-Premium*, *ADVENIR*, *économies*, *déjà là*
