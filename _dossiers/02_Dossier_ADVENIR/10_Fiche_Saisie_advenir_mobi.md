## Astuces avec Claude dans Edge

Si vous utilisez la barre latérale Claude dans Edge pendant la saisie :

1. **Ouvrir** ce document `10_Fiche_Saisie_advenir_mobi.pdf` dans un onglet
2. **Ouvrir** advenir.mobi dans un autre onglet
3. **Demander à Claude** dans la sidebar Edge :
   - *« Quelle est la valeur exacte du SIRET d'EGREENCITY'S à saisir ? »* → Claude lira ce PDF
   - *« Que dois-je mettre dans le champ Description du projet ? »* → Claude trouvera le paragraphe ÉTAPE 4
   - *« Coche les engagements opérateur correspondants à ma situation »* → Claude vérifiera la conformité

Claude dans Edge peut **lire le PDF affiché**, mais ne peut **pas remplir automatiquement** le formulaire pour des raisons de sécurité. Vous devez copier-coller manuellement, ce qui prend 30-45 minutes.
