## Comment utiliser ce dossier

### Étape 1 — Compléter les documents
1. Ouvrir `1-dossier-advenir-2026.md`, remplacer les `………` par vos données réelles
2. Ouvrir `2-chiffrage-advenir-bornes.xlsx` :
   - Vérifier le **barème** sur l'onglet "Bareme ADVENIR" (mettre à jour depuis advenir.mobi si besoin)
   - Compléter / modifier les sites prévus sur l'onglet "Chiffrage" (cellules **bleues**)
   - Ajuster le plan de financement sur l'onglet "Plan financement"
3. Ouvrir `3-checklist-pieces-administratives.md` et cocher au fur et à mesure

### Étape 2 — Rassembler les pièces administratives
Suivre la checklist (`3-...`). Compter ~3 semaines pour tout obtenir
(notamment Qualifelec, attestations URSSAF/fiscale, lettres d'intention mairies).

### Étape 3 — Envoyer les lettres d'intention aux 10 mairies
Adapter `5-lettre-intention-mairie.md` pour chacune des communes ciblées :
- Cayenne, Kourou, Saint-Laurent-du-Maroni, Matoury, Macouria,
- Rémire-Montjoly, Mana, Iracoubo, Roura, Sinnamary

### Étape 4 — Convertir les .md en PDF
Les `.md` peuvent être convertis en PDF via :
- VS Code + extension "Markdown PDF"
- Word (copier-coller le markdown puis sauvegarder en PDF)
- Pandoc : `pandoc 1-dossier-advenir-2026.md -o dossier.pdf`

### Étape 5 — Déposer sur advenir.mobi
1. Créer un compte sur https://advenir.mobi
2. Démarrer un dossier en ligne
3. Uploader les pièces dans l'ordre demandé
4. Joindre la lettre signée (`4-lettre-demande-advenir.md` au format PDF)
